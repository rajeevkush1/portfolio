## ğŸ¨ Customization

### Replace Placeholder Images

1. **Profile Image**: Replace `profile-placeholder.jpg` with your photo (recommended: 300x300px)
2. **Project Images**: Replace `project1-placeholder.jpg`, `project2-placeholder.jpg`, `project3-placeholder.jpg` with your project screenshots (recommended: 800x400px)

### Update Content

Edit `index.html` to update:
- Personal information
- Project details and links
- Skills and proficiency levels
- Education and certifications
- Social media links

### Customize Colors

Edit CSS variables in `styles.css`:

```css
:root {
    --primary: #6366f1;      /* Primary color */
    --secondary: #ec4899;    /* Secondary color */
    --accent: #8b5cf6;       /* Accent color */
    /* ... more variables */
}
```
